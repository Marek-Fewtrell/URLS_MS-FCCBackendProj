URL Shortener

How to use:

After the url enter the following:

  • /new/http://www.URL.com
  • /new/https://www.google.com
it will return the following object:
  • {"original_url": "http://www.URL.com", "short_url": "https://longing-vinyl.glitch.me/4321"}
  • {"original_url": "https://www.google.com", "short_url": "https://longing-vinyl.glitch.me/1234"}
